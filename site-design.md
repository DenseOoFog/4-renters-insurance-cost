# Site Design Blueprint (English)

## 1. Brand Positioning

**Site name:** RentersInsuranceCostGuide  
**Tagline:** Know the cost. Pick the right coverage. Rent smarter.  
**Audience:** US renters (18-40), students, first-time apartment renters.  
**Core promise:** Clear answers in 3 minutes: price, coverage, and next step.

## 2. UX Goals

- Let users find an answer in under 20 seconds.
- Keep page layout simple, fast, and mobile-first.
- Build trust with plain English, examples, and checklists.
- Create natural next-click paths for session depth and SEO.

## 3. Information Architecture

- Home
- Learn
  - What Is Renters Insurance
  - What It Covers / Does Not Cover
  - Deductible Explained
- Cost
  - Average Cost
  - Cost by State
  - Cost by City
- Claims
  - How to File a Claim
  - Theft / Water / Fire Guides
- For Your Situation
  - Students
  - Roommates
  - Pet Owners
- FAQ

## 4. Page Types

1. **Pillar page:** 1800-2500 words, table + TOC + FAQ.  
2. **Comparison page:** options table + pros/cons + decision tips.  
3. **State/City template:** intro + average price + factors + FAQ.  
4. **Checklist page:** printable steps for fast action.

## 5. Homepage Wireframe (Content Blocks)

1. Hero
   - H1: "How Much Is Renters Insurance in the US?"
   - Subtext: "Get fast, clear answers on cost, coverage, and claims."
   - CTA 1: Start with Cost
   - CTA 2: Learn Coverage Basics
2. Quick Answer Cards (3 cards)
   - Average monthly cost
   - Typical coverage range
   - Common deductible options
3. Top Guides (6 links)
4. Cost by State (search or A-Z list)
5. Claim Help Section
6. FAQ accordion (6 questions)
7. Trust block (editorial policy + update date)

## 6. Visual Style Guide

- **Primary color:** `#1D4ED8` (trust blue)
- **Secondary:** `#0F172A` (deep slate)
- **Accent:** `#F59E0B` (warning/highlight)
- **Background:** `#F8FAFC`
- **Body text:** `#334155`

- **Typography**
  - Headings: Inter / Poppins, semibold
  - Body: Inter / system-ui, regular
  - Base size: 17px desktop, 16px mobile

- **Spacing**
  - 8px spacing scale
  - Max content width 760px (articles)
  - Section gap 48-72px

## 7. Conversion + Adsense Layout Principles

- Keep first screen focused on answer, not ads.
- Article ads suggested positions:
  - after intro
  - mid-article (after section 2)
  - before FAQ
- Do not place ad blocks near fake buttons or table actions.
- Add "Last Updated" and "Sources" to improve trust.

## 8. 7-Day Design Execution

Day 1: finalize logo text + color + homepage structure  
Day 2: build homepage + top nav + footer  
Day 3: build article template page  
Day 4: publish first 3 pillar pages  
Day 5: build state template page  
Day 6: internal links + schema + performance checks  
Day 7: mobile QA + submit sitemap
