# 4-renters-insurance-cost

这是一个面向美国租客的 `Renters Insurance` 信息型站点，用 SEO 获取流量，再用 Adsense 变现。

## 1) 文件夹里建议有的内容

- `README.md`：项目定位、执行节奏、内容框架（本文件）。
- `keywords.md`：关键词库（按意图分组）。
- `content-calendar-30d.md`：前30天发布计划（每天1篇）。
- `article-template.md`：单篇文章标准结构（避免写散）。
- `pages/`：已发布文章草稿或最终稿。
- `assets/`：图表、清单图、封面图等素材。

## 2) 网站核心定位（一句话）

帮首次租房人群快速搞懂：要不要买租客保险、怎么买更省钱、出险如何理赔。

## 3) 目标用户

- 第一次在美国租房的年轻人/留学生。
- 被房东要求提供保险证明（COI）的人。
- 预算敏感、想知道“最低多少钱能合规”的租客。

## 4) 内容支柱（先做这6类）

1. 基础认知：Renters Insurance 是什么、保什么不保什么。  
2. 费用价格：每月多少钱、不同州/城市差异。  
3. 购买决策：怎么选保额、deductible 怎么定。  
4. 理赔流程：被盗、漏水、火灾时怎么办。  
5. 房东要求：租约要求保险时如何满足。  
6. 省钱技巧：折扣、捆绑、提高自付额是否划算。  

## 5) URL/栏目结构建议

- `/what-is-renters-insurance/`
- `/renters-insurance-cost/`
- `/renters-insurance-by-state/`
- `/how-much-coverage-do-i-need/`
- `/deductible-explained/`
- `/landlord-requires-renters-insurance/`
- `/how-to-file-a-renters-insurance-claim/`
- `/renters-insurance-for-students/`
- `/renters-insurance-faq/`

## 6) 前30天最小可执行计划

- 第1周：发布 5 篇基础与价格文章（先抢核心词）。
- 第2周：发布 7 篇决策与理赔文章（提升商业意图）。
- 第3周：发布 9 篇场景长尾（学生、宠物、合租、公寓类型）。
- 第4周：发布 9 篇州/城市类模板文章（可批量扩展）。

目标：30天至少上线 30 篇，形成内链网络。

## 7) 每篇文章固定结构（强制）

1. 先给结论（60-100字）。  
2. 用表格回答核心问题（价格/保额/步骤）。  
3. 展开解释（3-5个小节）。  
4. 常见错误（至少3条）。  
5. FAQ（4-6问）。  
6. 行动建议（下一步做什么）。  

## 8) Adsense 友好原则

- 信息密度高但不堆砌广告。
- 不诱导点击广告，不做“像按钮”的误导位。
- 正文解决问题优先，广告只做补充。
- 先做流量和停留，再优化广告位。

## 9) 你现在就可以做的三件事

1. 先写 3 篇：  
   - What Is Renters Insurance  
   - How Much Does Renters Insurance Cost  
   - Is Renters Insurance Worth It  
2. 给每篇加 4 个内链。  
3. 提交 Search Console 并请求收录。
